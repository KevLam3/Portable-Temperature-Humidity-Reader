# About
This is a temperature & humidity reader that can be placed anywhere. This is currently a work in progress.

# Hardware
- Arduino Uno
- Breadboard
- Wires (both M-M and M-F)
- 10K Potentiometer
- Power Supply Module
- Any DHT Module (the ones shown a 3 pins DHT11)
- Button Module
- Resistors (10K & 100)
- LCD1602 Module
- HC-05 Module
